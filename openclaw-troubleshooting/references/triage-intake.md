# 信息收集入口

## 最小必要信息采集（只问 4 件事）

### 1. 最近做了什么变更？
- 升级 OpenClaw 版本？
- 修改配置文件？
- 安装新插件？
- 更改网络设置？
- 其他系统变更？

### 2. 现在的症状是什么？
- 安装失败？
- 服务无法启动？
- Gateway 访问不了？
- 消息通道不响应？
- 模型调用出错？
- 其他具体现象？

### 3. OpenClaw 版本和安装方式？
- 版本：`openclaw --version` 输出
- 安装方式：npm / git / installer / 其他
- 操作系统：Windows / macOS / Linux（具体版本）

### 4. 有没有明确报错或日志片段？
- 错误信息文本
- 日志片段（已脱敏）
- 截图描述（如有）

---

## 状态记录建议

在每轮对话末尾总结当前诊断状态，例如：

> *当前状态：版本 v1.2 | 本地部署 | Telegram 不响应 | 最近刚升级*
