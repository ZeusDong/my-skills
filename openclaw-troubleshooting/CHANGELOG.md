# Changelog

All notable changes to `openclaw-troubleshooting` will be documented in this file.

## v0.1.0 - 2026-03-15
### Added
- Initial release of `openclaw-troubleshooting` skill.
- Independent troubleshooting workflow that does not rely on a healthy OpenClaw runtime.
- Security-first triage and escalation boundaries.
- Official CLI command ladder for interactive diagnosis.
- Topic-based references for install/startup, gateway/network, channels, models, updates, and diagnostics.
- Optional helper scripts for context collection, JSONL parsing, and redaction bundle generation.
